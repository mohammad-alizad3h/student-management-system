# Student Management System

A simple **console-based student management system** written in Python.  
This project allows you to **add, view, edit, delete, search, and sort students** with a colorful console interface.

## Features

- Add new students with name and age ✅
- View students in a **colorful table** ✅
- Edit existing student information ✅
- Delete students **securely with confirmation** ✅
- Search students by **name or number** ✅
- Sort students by **name or age** ✅
- Works on **Windows, macOS, Linux** with Python 3 ✅
